# E-Commerce-Executive-Dashboard-PowerBI
Interactive Power BI dashboard analyzing revenue, customers, discounts, and product performance.

##  Project Overview
This project presents an interactive executive dashboard built using Power BI to analyze key performance metrics of an e-commerce business.
The dashboard focuses on revenue trends, customer insights, product performance, and discount impact to support data-driven decision-making.

##  Data Preparation
* Performed comprehensive data cleaning and transformation using Power BI
* Handled missing values, removed duplicates, and corrected inconsistent data entries
* Standardized column formats and data types to ensure accuracy
* Created calculated columns and applied necessary transformations for reporting
* Structured and optimized the dataset to support efficient data modeling and analytical reporting

##  Objectives
* Identify top-performing cities
* Analyze monthly revenue trends
* Determine best-performing product category
* Evaluate discount impact on revenue
* Identify top 5 customers

##  Key KPIs
* Total Revenue: 69.53M
* Total Orders: 0.57K
* Total Customers: 0.55K
* Average Discount: 93.86%

##  Key Insights
* Pune generates the highest revenue
* Electronics is the top-performing category
* January recorded peak sales
* Higher discounts show a positive impact on revenue
* A small group of customers contributes significantly to total revenue
